Frequently asked questions

General

How to account for funnel code of MAGIC mating design?

If function code is known for each sampled offspring, then split samples into subpopulations, each having different funnel codes. If function code is unknown, set isfglexch=true in offspringinfo so that founders are assumed a priori exchangeable.

How to set keyarg for genetic map construction?

  • It is strongly recommended to set ncluster for the number of chromosomes (linkage groups). If ncluster is not known, set the range by minncluster and maxncluster

  • Keyargs minlodcluster, ncomponent, knncluster, and clusteralg are for marker grouping, and keyargs knnorder and minlodorder for marker ordering

  • The quality of marker grouping is indicated by a large eigengap in the eigen.png plot and a large silhouette score. If repeatedly run MagicMap.magicmap, pairwise LD and linkage analysis will be skipped.

  • For accurately spacing markers, run MagicImpute.magicimpute by setting isspacemarker=true and specifying mapfile as the constructed mapfile of magicmap.
